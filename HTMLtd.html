Introduction

HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It defines the meaning and structure of web content.

You need more than just HTML to create an appealing website, but that will only be introduced later in the document(see part 4).

Basic Structure

Every HTML document created has a basic template or structure that it must follow. In this structure it will include mostly non-visual elements. These elements help browser search effiency and capability. This is also where you link your HTML and CSS files.

{!DOCTYPE html}
{html}
{head}
{meta}
{/head}
{/html}
m

{meta charset="utf-8"}
{meta name="viewport" content="width=device-width, initial-scale=1.0"}

h elements

h elements in HTML act as headers or titles for your page. There are 7 predefined h elements each with different sizes, but you can customize each of them individually if you use CSS. You would generally use h elements as a title before text

Here's an example(in actual HTML the title would have a larger font and be bold)

{h}Placeholder title{/h}
//you would have text that follows here//
//this is where the {p} element comes into play//
p elements

p elements in HTML serve as paragraphs or text boxes. this is where you would place the bulk of your text on your page whether it's a description of an item, or a introduction to your page.

Here is an example of what that would look like(the p element will have a smaller font size than the h element)

{h}Placeholder title{/h}
{p}this is what text in a p element would look like. it would usually come after the h element{/p}
CSS and styling

HTML is just used to create the basic foundation for your website so that you can start to build and alter it the way you want to. Thats where CSS comes into play. CSS,Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML.

CSS is often much simpler than HTML, here is an example of what some of the CSS for this page looks like
